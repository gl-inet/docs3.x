# GL.iNetへようこそ

GL.iNetは、ハードウェアをオープンソースソフトウェアと簡単に連携させるための会社です。このドキュメントサイトは、オープンソースソフトウェアがハードウェア上でどのように機能するのかを新人にアドバイスすることを目的としています。製品が進化するにつれて、このドキュメントサイトは随時更新されます。
