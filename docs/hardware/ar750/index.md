#  GL-AR750 トラベル AC ルーター



最初のデュアルバンドACトラベルルーターです。もう一つのイーサネットポート（全部で三つ）が追加しております。最大128GBのマイクロスロットが持っています。

PoEオプションがありますし、手頃な価格WiFiのカバレッジのための最良の選択です。

2.4GアンテナがPCBでありますし、uFLコネクタがありません。しかしながら、5GアンテナがuFLを持っています。
